# 📈 ForexPulse — Live Trading Signals

A beginner-friendly forex trading signal dashboard with **real live market prices**, clear Top 5 Buy/Sell recommendations, and a simulated portfolio tracker.

**⚠️ Educational tool only — not financial advice.**

## 🌐 Live Demo

👉 **Open the dashboard:** Just visit your GitHub Pages URL after deploying!

Or download `index.html` and double-click to open in your browser.

## ✨ Features

- **📡 Real Live Prices** — Fetches actual forex rates from the market every time you open it
- **🟢 Top 5 BUY NOW** — Best buying opportunities ranked by confidence score
- **🔴 Top 5 SELL NOW** — Best selling opportunities ranked by confidence score
- **📊 Each Signal Shows:**
  - Entry price, profit target, and stop-loss
  - Confidence meter (0-100%)
  - Expected profit in pips
  - Simple "Why?" explanation
- **💼 Simulated Portfolio** — Practice with $10,000 virtual money
- **🎓 Beginner-Friendly** — 4-step guide, educational tooltips, risk warnings
- **🌙 Professional Dark Theme**
- **📱 Responsive** — Works on desktop and mobile

## 📊 How Signals Work

The dashboard analyzes 8 major forex pairs using three technical indicators:
- **RSI** (Relative Strength Index) — Detects overbought/oversold conditions
- **MACD** — Identifies momentum shifts and trend changes
- **Moving Averages** — Shows overall trend direction

Each pair gets a confidence score (0-100%) based on how many indicators agree. Higher score = stronger signal.
